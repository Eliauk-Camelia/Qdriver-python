# 电赛云台视觉跟踪 — 任务逻辑

## 总前提

设备上电初始化：
- 激光发射
- 电机使能
- 全局变量 `problem` 记录运行状态（1 / 2 / 3），分三种分支逻辑

---

## 第一问：problem = 1（寻靶 + 停住）

```
while True:
    1. 读取图像帧
    2. 颜色阈值二值化（筛选目标颜色）
    3. 查找画面中最大矩形轮廓
    4. 分支判断：
       ├─ 无目标矩形 → 云台 Yaw 轴以 yaw_speed 持续转动（扫描）
       └─ 有目标矩形 → 停止转动，保持静止
```

关键点：只需 Yaw 轴转动扫描，找到目标就停，不需要跟踪。

---

## 第二问：problem = 2（跟踪 + 上报）

```
while True:
    1. 读取图像帧
    2. 颜色阈值二值化
    3. 查找最大矩形轮廓
    4. 分支判断：
       ├─ 无目标矩形 → Yaw 轴以 yaw_speed 转动
       └─ 有目标矩形 →
           ① 计算误差 err_x = cx - w/2, err_y = cy - h/2
           ② 误差值发送至上位机（串口上报 / 显示）
           ③ 基于误差做云台跟随控制（PID → speed_ctrl）
```

关键点：找到后持续跟踪，同时把误差上报给裁判系统/上位机。

---

## 第三问：problem = 3（跟踪 + 激光画圆）

```
while True:
    1. 读取图像帧
    2. 颜色阈值二值化
    3. 查找最大矩形轮廓
    4. 分支判断：
       ├─ 无目标矩形 → Yaw 轴以 yaw_speed 持续转动
       └─ 有目标矩形 →
           ① 计算误差 err_x, err_y
           ② t += dt  （时间累加）
           ③ 叠加画圆偏移：
              cmd_x = err_x + R * cos(ω * t)
              cmd_y = err_y + R * sin(ω * t)
           ④ 下发给云台做跟随控制
```

关键点：
- err 让激光指向靶心，cos/sin 让激光绕靶心画圈
- R（圆周半径，像素）和 ω（角频率）需要调参
- ω 太快电机跟不上，圈会变形
